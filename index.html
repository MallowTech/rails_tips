Contribution

To submit a tip to the list, fork the repository and add your tip to a new file in the rails_tip folder. The name of the file should be YYYY-MM-DD-tip-name.

Use the format mentioned in repository when writing your tip. Your tip should be brief. You may add links to other sites or videos that give more insight if you wish.

Once your tip is ready, issue a pull request with the PR template and your tip will be reviewed. Every day one tip will be merged from the available pull requests.
